Browser game

A browser game is a video game that is played on the internet using a web browser. They are sometimes referred to more specifically by their format, such as Flash games or HTML5 games. They are generally free-to-play and can be either single-player or multiplayer. It is not necessary to install a browser game; simply visiting the webpage will run the title in a browser. Some browser games were also made available as mobile apps, PC games, or console titles. More at Wikipedia
